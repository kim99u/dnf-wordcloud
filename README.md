# dnf-wordcloud
던전앤파이터 홈페이지 크롤링 및 워드클라우드 생성

## 프로젝트 소개
던전앤파이터 홈페이지 공식 게시판에 작성된 게시물들의 제목을 크롤링하여 가장 많이 쓰인 단어들로 워드클라우드를 생성한다.



## 개발기간
최초 개발 : 2017.10 ~ 2017.12 <https://blog.naver.com/rladb0115/221160144592>

이후 수정 : 2023.06



## 개발 환경
python 3.10

Java 17

Vscode
